Web Project 2 – ASP.NET Web Forms (C#)

This repository contains a university web development project built using ASP.NET Web Forms with C#.

What’s inside

  • Multiple pages (e.g., Home.aspx, Contact.aspx, CV.aspx)
  • Master Page layout (.Master)
  • Server-side logic in .aspx.cs
  • Configuration files (Web.config)

Note: GitHub Pages can’t run .aspx projects. To run this project, open it in Visual Studio and start it with IIS Express.
